## Delegation Map

Use specialist skills for:

- Program development: Solana dev skill
- Anchor and Pinocchio implementation: core development skill
- Security audits: security/auditor skill
- DeFi protocol APIs: official protocol skills when available
- Legal and compliance: crypto legal skill or qualified professional
- Formal verification: formal verification skill
- Infrastructure deployment: infra/deployment skill

This founder skill should integrate those outputs into market, proof, and adoption strategy. It should not replace them.
